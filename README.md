# 🔐 Password Generator (June 28th - July 1st, 2026.)

A simple terminal-based password generator built with Python.

This wasn't meant to be some huge project. After spending the past month living inside HTML and CSS, I figured it was time to make sure I still remembered how to write Python... before I accidentally started writing `<div>`s inside `.py` files.

I actually started this on my birthday. Instead of diving into another frontend project, I wanted to build something a little more relaxed for the day, a small Python refresher that I could casually chip away at over the next few days.

Turns out I still can.

Built casually over a few days, adding a little bit whenever I felt like coding.

---

## 🚀 Features

* Generate random 10-digit passwords
* Create your own custom password
* View your currently saved password
* Simple interactive terminal menu
* Clean function-based structure
